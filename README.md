# Global Superstore Sales Dashboard in Excel

## Overview

This project is an interactive Excel dashboard built using the **Global Superstore dataset** from Kaggle. It analyzes retail business performance through data cleaning, lookup functions, pivot tables, charts, slicers, and KPI cards.

## Project Objectives

* Analyze sales trends over time
* Compare category and segment performance
* Identify top-performing products
* Evaluate profit across regions
* Build an interactive business dashboard in Excel

## Dashboard Features

* **KPI Cards**

  * Total Sales
  * Total Orders
  * Total Profit
  * Total Quantity

* **Interactive Charts**

  * Sales by Month
  * Sales by Category
  * Profit by Region
  * Sales by Segment
  * Top 10 Products by Sales
  * Order Priority Analysis

* **Slicers**

  * Year
  * Region
  * Segment
  * Category

## Excel Skills Used

* Data Cleaning

  * TRIM
  * CLEAN
  * PROPER
  * Date formatting
  * Helper columns (Year, Month)

* Lookup Functions

  * VLOOKUP
  * INDEX + MATCH

* Analysis & Visualization

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * KPI Cards
  * Dashboard Design

## Dataset

Global Superstore dataset from Kaggle

## Outcome

Built an interactive business dashboard to analyze retail sales trends and improve Excel skills for beginner-level data analyst roles.
